# 10个精选心理学故事（可直接做短视频）
### 1. 沉默成本效应
一个人越付出，越舍不得放手。
不是多爱对方，是不甘心自己之前的时间、情绪、真心白白浪费。
很多关系走不出来，不是放不下人，是放不下自己的付出。

### 2. 登门槛效应
别人帮了你一个小忙，就更容易帮你大忙。
想让一个人信任你，先让他帮你一件小事，好感会瞬间提升。

### 3. 损失厌恶
人不怕得到，最怕失去。
比起获得快乐，人们更害怕痛苦，所以失去的痛苦，永远大于得到的快乐。

### 4. 墨菲定律
越怕什么，越来什么。
你越焦虑、越担心、越内耗，坏事越容易发生，心态稳，事才顺。

### 5. 刺猬效应
人和人要保持距离。
靠太近会扎伤彼此，离太远会生疏，舒服的关系，都是不远不近。

### 6. 羊群效应
别人做什么，你就跟着做什么，很容易迷失自己。
大多数人走的路，不一定是对的，只是走的人多。

### 7. 皮格马利翁效应
你相信什么，就会吸引什么。
你觉得自己幸运，就会遇到好事；你总自我否定，生活就越来越糟。

### 8. 超限效应
话不要说太多，事不要做太满。
越是反复讲道理，别人越反感；点到为止，最让人记住。

### 9. 旁观者效应
人越多，越没人帮忙。
一个人求助，别人会帮；一群人在场，人人都观望。

### 10. 习得性无助
长期否定自己、被打压，慢慢就不敢努力、不敢争取。
不是你不行，是你被自己困住了。
